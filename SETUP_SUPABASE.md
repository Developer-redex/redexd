# إعداد قاعدة بيانات Supabase

لربط موقعك بقاعدة بيانات حقيقية (Supabase) وجعله يعمل "أونلاين"، اتبع الخطوات التالية:

## 1. إنشاء مشروع جديد
1. اذهب إلى [supabase.com](https://supabase.com) واعمل تسجيل دخول (Sign Up).
2. اضغط على **"New Project"**.
3. اختر اسماً لمشروعك (مثلاً `Redex`) وكلمة مرور قوية لقاعدة البيانات.
4. انتظر قليلاً حتى يتم إنشاء المشروع.

## 2. إنشاء الجداول
1. بعد فتح المشروع، اضغط على أيقونة **SQL Editor** في القائمة الجانبية (شكلها ورقة وعليها قلم).
2. اضغط على **"New query"**.
3. انسخ الكود الموجود في ملف `supabase_schema.sql` (موجود في مجلد مشروعك) والصقه في المحرر.
4. اضغط على زر **RUN** الأخضر (أسفل اليمين).
5. ستظهر رسالة "Success" مما يعني أنه تم إنشاء الجداول بنجاح.

## 3. ربط الموقع
1. اذهب إلى إعدادات المشروع في Supabase (**Project Settings** -> أيقونة الترس).
2. اختر **API** من القائمة الجانبية.
3. ستجد معلومات **Project URL** و **anon public key**.
4. انسخ هذه المعلومات وضعها في ملف `scripts/api.js` مكان النصوص المخصصة لذلك:
   ```javascript
   const SUPABASE_URL = 'ضع رابط مشروعك هنا';
   const SUPABASE_KEY = 'ضع المفتاح anon key هنا';
   ```

## 4. استضافة الموقع على GitHub
1. بعد ربط قاعدة البيانات، قم برفع ملفات الموقع على GitHub.
2. فعل خيار **GitHub Pages** من إعدادات المستودع (Settings -> Pages).
3. سيصبح موقعك متاحاً للجميع "أونلاين" ويعمل مع قاعدة البيانات الحقيقية!
